# Endpoints

## Find Competitors

```
GET /find-competitors?domain=EV%20Mobility&region=Bangalore
```

payload:

```
{
    "domain": "EV Mobility",
    "region": "Bangalore"
}
```

---

## Get Competitors

```
GET /get-competitors
```

Returns the stored competitors from the last search.
